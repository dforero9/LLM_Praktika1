# Proyecto web HTML: Getafe como destino de viaje

Proyecto estático preparado para subir a un repositorio Git.

## Contenido

- `index.html` → página principal.
- `styles.css` → estilos visuales inspirados en portales de reservas (sin copiar marca ni activos protegidos).
- `script.js` → pequeño script para mostrar el año actual.
- `.gitignore` → archivo básico para repositorios.

## Personalización rápida

Busca `Alumno X` en `index.html` y sustitúyelo por el nombre que necesites.

## Cómo usarlo

1. Descomprime el proyecto.
2. Abre `index.html` en el navegador.
3. Si quieres subirlo a Git:

```bash
git init
git add .
git commit -m "Proyecto inicial web de Getafe"
```

## Fuentes usadas para el contenido

- Ayuntamiento de Getafe
- VisitMadrid
- Comunidad de Madrid
- Wikipedia / Wikimedia Commons

## Nota de diseño

La estética está inspirada en webs de viajes y reservas: cabecera azul, tarjetas, bloques visuales y llamadas a la acción.
